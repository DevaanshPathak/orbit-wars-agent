# v5 Notes

v5 adds model-ready candidate features, a bounded deep planner, and Hugging Face artifact upload for offline training outputs.

The single v5 training notebook trains, evaluates, saves graphs under `exports/graphs/`, and uploads the full export folder to `devaanshpa/orbit-wars-agent/v5`.

Artifacts generated by v5 notebooks must stay out of git. Later local downloads from Hugging Face should go into root `models/`.

Training CSVs should be generated with `generate_training_data.py`; each command writes one run-start folder under root `data/` and uploads to Hugging Face under `data/<run_start_timestamp>/`.
